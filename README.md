# 💳 ATM Management System (Java)

## 📌 Project Overview

The ATM Management System is a Java-based web application that simulates basic banking operations.
It allows users to securely perform ATM transactions such as deposit, withdrawal, and balance checking.

---

## 🚀 Features

* 🔐 User Authentication (Login System)
* 💰 Balance Inquiry
* ➕ Deposit Money
* ➖ Withdraw Money
* 🔄 Fund Transfer
* 📜 Transaction History

---

## 🛠️ Technologies Used

* ☕ Core Java
* 🌐 Advanced Java (JSP & Servlet)
* 🗄️ JDBC (Java Database Connectivity)
* 🛢️ MySQL Database
* 💻 HTML, CSS

## ⚙️ How to Run the Project

1. Clone the repository

   ```bash
   git clone https://github.com/your-username/atm-management-system.git
   ```

2. Open the project in an IDE (Eclipse / IntelliJ)

3. Setup Database

   * Import `atm.sql` file into MySQL

4. Configure JDBC

   * Update database username and password

5. Run on Server

   * Start Apache Tomcat server

6. Open in Browser

   ```
   http://localhost:8080/ATM-System
   ```
